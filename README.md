# IOC Enrichment Tool

Python tool for IOC enrichment using VirusTotal API.

## Features

- IP reputation lookup
- File hash lookup
- VirusTotal integration
- Bulk IOC enrichment
- File hash analysis

## Usage

```bash
python ioc_lookup.py
```

## Requirements

```bash
pip install requests
```
### Bulk IOC Lookup

```bash
python bulk_ioc_checker.py
```

### File Hash Reputation Check

```bash
python hash_checker.py
```
